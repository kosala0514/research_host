Data Types

Python supports several data types, each with its own characteristics and uses.

  • Text Type

    • str - Represents a sequence of characters enclosed in single quotes ('') or double quotes ("").

  • Numeric Types

    • int - Represents whole numbers, such as 1, -5, or 1000.

    float - Represents decimal numbers, such as 3.14 or -2.5.

    complex - Represents numbers with both real and imaginary parts, such as 2 + 3j or -1 - 4j.

  • Sequence Types

    • list - Represents an ordered collection of items enclosed in square brackets. Lists can contain elements of different types and are mutable (modifiable).

    tuple - Represents an ordered collection of items enclosed in parentheses (()). Tuples are similar to lists, but they are immutable (cannot be modified).

  • Mapping Type

    • dict - Represents a collection of key-value pairs enclosed in curly braces ({}). Keys are unique and used to access corresponding values. Dictionaries are mutable.

  • Set Type

    • set - Represents an unordered collection of unique elements enclosed in curly braces ({}). Sets do not allow duplicate values and support set operations like union, intersection, and difference.

    frozenset - Similar to a set but is immutable.

  • Boolean Types

    • bool - Represents truth values, either True or False. It is often used for logical operations and conditions.

  • Binary Types

    • bytes - The bytes data type represents an immutable sequence of bytes. It is used to store binary data in a fixed-size format. The individual bytes in a bytes object can be accessed using indexing, similar to a string. However, unlike strings, bytes objects are immutable.

    bytearray - The bytearray data type is similar to bytes, but it represents a mutable sequence of bytes. It allows you to modify individual bytes or perform in-place operations.

    memoryview - The memoryview data type provides a way to access the internal data of other binary objects (like bytes or bytearray) without making a copy. It allows for efficient manipulation of large amounts of binary data.

  • None Type

    • NoneType - Represents the absence of a value or a null value. It is often used to indicate the absence of a return value or as an initial/default value.
