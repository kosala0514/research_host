Tuple

One of the 4 built-in data type that can use to store multiple items in a single variable.

This is ordered, unchangeable and allow duplicate

Example:-

To get length of the tuple

create tuple with one item


Access tuple item

We can access item in the tuple by referring index

We can access item from the end of the tuple by referring negative index.

We can access range of items by specifying start and end point of the range.

We can check whether the item is exist or not in th tuple

Change tuple items

We can change the item of tuple converting to list

Unpack tuple

we are also allowed to extract the values back into variables. This is called "unpacking"

If the number of variables is less than the number of values, you can add an * to the variable name and the values will be assigned to the variable as a list

We can use for loop
