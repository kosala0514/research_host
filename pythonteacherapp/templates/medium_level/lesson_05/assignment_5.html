Scenario :

Write a Python function called calculate that takes three arguments: a, b, and operation. The operation argument is a string that specifies which mathematical operation to perform on a and b. The function should return the result of the specified operation.

The possible values for operation are:

  • "add": returns the sum of a and b.
  • "subtract": returns the difference between a and b (i.e., a - b).
  • "multiply": returns the product of a and b.
  • "divide": returns the result of dividing a by b.

If operation is not one of these four strings, the function should return the string "Invalid operation".Also You can assume that a and b will always be numeric values, but you should handle division by zero by returning the string "Cannot divide by zero".
